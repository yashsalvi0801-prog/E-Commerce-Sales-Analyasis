# E-Commerce-Sales-Analyasis

Created an interactive Dashboard to track and analyse online sales data.
Used complex parameters to drill down in the worksheet and customisation using filters and slicers.
Created a connection, joined new tables, calculated to manipulate data and enables user driven parameters for
visualisations.
Used different types of customised visualisation (Bar-Chart, Pie-Chart, Doughnut-Chart, Clustered Bar Chart,
Scatter-Chart, Line-Chart, Area-Chart, Map, Slicers Etc).
